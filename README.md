# Real-Time Website Spam Filter 🚀

## 📌 Overview
This project is a rule-based real-time spam detection system that monitors website form submissions, detects spam using predefined rules, and stores data for analysis.

---

## 🎯 Objective
- Detect spam messages from website forms
- Analyze user behavior (fast submission, repeated inputs, etc.)
- Store data in MySQL database
- Visualize insights using dashboard

---

## 🛠️ Technologies Used
- Python (Pandas, NumPy)
- Selenium (for data fetching)
- MySQL (database)
- Streamlit (dashboard)
- Matplotlib (visualization)

---

## ⚙️ Features
- Real-time form data collection using Selenium
- Rule-based spam detection
- Stores data in MySQL
- Dashboard showing:
  - Total submissions
  - Spam / Ham / Suspect
  - Trends & analysis
